# Tarea
Planificar la Conversion de los mockups que están en STITCH en el proyecto llamado Advanced PDF Editor con id: 217974861917299723 (puedes usar el mcp de Stich que tienes instalado) a una web funcional en Node.

En una aplicación web que manipula PDF al estilo Adobe Acrobat pero sin tantas funcionalidades.

# Especificaciones
El proyecto usará Node en su version 22

Las funcionalidades serán:

- [ ] Importar pdfs, ya sean de una página o varias
- [ ] Poder añadir más pdfs
- [ ] Listado de miniaturas de cada pagina
- [ ] Reordenar las paginas por medio de arrastrar y soltar o marcando el numero de pagina de destino, puedes reordenar multiples a la vez
- [ ] Eliminar una o varias paginas
- [ ] Comprimir pdf
- [ ] Buscar texto dentro del pdf
- [ ] Añadir/Eliminar/Editar Texto dentro del pdf
- [ ] Cubrir los espacios dentro de un formulario pdf
- [ ] No se necesitará login
- [ ] Dark mode
- [ ] Poder exportar todo el pdf resultante o solo las paginas deseadas por medio de "," o "-"
- [ ] Posibilidad de aumentar o disminuir las miniaturas de las paginas del pdf
